A note about the writing on this blog

When I write "It is unprofessional to write code without tests", please read "In my opinion, based on my extremely limited knowledge of the world and my many conscious and unconscious biases, writing code without tests is unprofessional in cases without extenuating circumstances."

I write the former because it's faster and reads better. If it seems like I'm being an arrogant know-it-all about something, please just expand my statements into something more like the latter.
